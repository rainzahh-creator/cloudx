Cài đặt API key

Lấy key miễn phí tại aistudio.google.com/apikey. Key có thể ở định dạng cũ AIzaSy... hoặc định dạng mới AQ.... — cả hai đều dùng được.

Key chỉ lưu trong trình duyệt của bạn (localStorage), không gửi lên máy chủ CloudX. Ai xem mã nguồn trang lúc bạn đang mở đều có thể lấy được key — cân nhắc dùng backend proxy nếu triển khai công khai.
